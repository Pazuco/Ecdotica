# Ecdotica

## Descripción

Ecdotica es un proyecto dedicado a la ecdótica digital, la ciencia que estudia la transmisión de textos y su edición crítica. Este repositorio proporciona herramientas y recursos para el análisis y edición de textos históricos y literarios.

## Características

- Herramientas para análisis textual
- Soporte para edición crítica
- Gestión de variantes textuales
- Interfaz para comparación de manuscritos

## Instalación

```bash
git clone https://github.com/Pazuco/Ecdotica.git
cd Ecdotica
```
